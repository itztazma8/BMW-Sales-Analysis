# BMW Sales Analysis
The goal of this project is to show different insights based on the sales of 2018-2025 for BMW. The story goes into how different regions perform in case of sales. Analysis of different models is also shown here. All the necessary links are given below.
<br>
**Dataset**: [Sales](https://www.kaggle.com/datasets/dmahajanbe23/bmw-global-automotive-sales)
<br>
**Tableu Link**: [Tableu](https://public.tableau.com/views/BMWSalesAnalysis_17732501064220/GlobalBMWRevenueAnalysis2018-2025?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)
